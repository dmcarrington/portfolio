Recent Projects & Proof of Work

Actively building and shipping. Here is what I am working on right now.

Unfiltered

A Nostr photo-sharing Android app. Native Kotlin with Jetpack Compose, rust-nostr bindings, Blossom image hosting, Lightning zaps, and Amber key management. 6 releases shipped, 105 commits.

View on GitHub

Zap Gallery

A decentralised photo marketplace built with SvelteKit. Sell photos for zaps using Nostr Wallet Connect, with AES-256-GCM encrypted originals stored on Blossom and keys delivered via NIP-04 DMs.

View on GitHub

Time Tribe

A cyberpunk-themed social Pomodoro timer for Android. Publishes focus sessions to Nostr relays, with friend leaderboards, contribution matrices, and Blossom avatar uploads. Kotlin, Jetpack Compose, Hilt DI.

View on GitHub

Bitwatch

A Bitcoin transaction signing device disguised as a smartwatch. ESP32-based, supporting PSBT signing, QR code address display, and WiFi portal for transaction management. Hidden in plain sight.

An interactive TypeScript demonstration of brain wallet key derivation, deployed on Vercel. Built for educational use at meetups and workshops.
